# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-05-28

### Added
- **统计页面重设计**：历史页面「趋势」tab 改为「统计」，新增日期范围选择器（近一周/近一个月/近半年/近一年/自定义）
- **分离图表展示**：统计页拆分为三块 — 日均值卡片（热量/碳水/蛋白/脂肪）、卡路里柱状图、营养素分组柱状图
- **拍照识别入口**：食物页面搜索栏右侧新增扫描图标，点击可拍照识别营养成分表
- **弹窗内拍照**：新增/编辑食物弹窗顶部新增「拍照识别」按钮，识别后直接回填表单
- **多 AI 提供商支持**：支持智谱 GLM-4V、Kimi (Moonshot)、Gemini 三家 AI 视觉 API，优先级智谱 > Kimi > Gemini
- **API Key 测试**：设置页面每个 API Key 输入框旁新增「测试」按钮，可验证连通性
- **网络诊断**：设置页面新增「诊断网络连通性」按钮，检测各 AI 服务端点可达性

### Changed
- **OCR 方案升级**：从 Tesseract.js 正则匹配升级为 AI 视觉识别，准确率大幅提升
- **图片压缩**：拍照识别时自动压缩图片至 1920x1920 JPEG 0.8 质量，避免 API 400 错误
- **错误提示优化**：智谱 403 错误提示额度/频率限制原因，Gemini 429 提示配额用完

### Removed
- **移除 Claude API**：因 Anthropic 不支持浏览器直连（CORS 限制），移除 Claude 选项
- **移除旧趋势图**：删除 `TrendChart.vue`，替换为新的 `StatsChart.vue`

### Fixed
- **测试与保存不一致**：修复「测试」按钮读取旧 localStorage 值导致成功但「保存」后失效的问题
- **图片过大报错**：拍照时限制最大尺寸并压缩，避免 base64 过大导致 API 拒绝

## [0.1.0] - 2026-05-28

### Added
- **日期导航**：今日页面顶部日期支持左右箭头切换，点击日期弹出日期选择器（年/月/日滚动列 + 「今天」快捷按钮）
- **条码扫描**：新增 `BarcodeScanner` 组件，支持扫条码和拍配料表两种模式
- **营养成分表 OCR**：拍照识别营养成分表，自动提取碳水/蛋白/脂肪数据（基于 Gemini Vision API）
- **中文条码查询**：多源级联查询（OpenFoodFacts → UPCitemdb → mxnzp.com），提升 69 开头中国商品条码覆盖率
- **最近食物 tab**：食物选择器新增「最近」tab，显示最近添加的食物（按时间倒序）
- **长按多选**：今日页面和历史详情页支持长按条目进入多选模式，可批量复制/保存为食谱/删除
- **单击编辑**：非多选模式下单击条目可快速编辑份数和餐次
- **餐次占比设置**：设置页面支持自定义力训日/休息日各餐次的碳水和蛋白质占比
- **分类排序**：食物页面排序模式支持调整分类顺序（置顶/上移/下移）
- **食物排序**：每个分类内的食物支持独立排序（置顶/上移/下移），排序数据持久化到 localStorage
- **自定义分类**：支持新建自定义食物分类
- **导出/导入**：设置页面支持导出 JSON 备份（包含食物/食谱/日志/排序/分类/占比），支持导入恢复
- **历史页面**：日历热力图（绿=达标/灰=不足/红=超标）+ 趋势图（碳水/蛋白/脂肪折线 + 热量柱状）
- **历史详情**：点击日历日期进入详情页，显示当日所有条目，支持编辑/删除/切换训练日类型
- **餐次分组**：今日页面和历史详情页按餐次分组显示（早餐/午餐/晚餐/加餐），显示各餐次目标占比
- **代谢表盘**：今日页面顶部圆环显示剩余热量，卡片显示碳水/蛋白/脂肪进度和每公斤体重倍数
- **PWA 支持**：支持安装为 PWA 应用，自定义图标（三色宏量环 + 刀叉）
- **预置食物库**：内置 200+ 常见食物数据（主食/蔬菜/水果/肉蛋奶/零食等）

### Changed
- **UI 优化**：今日页面头部轻量化设计，圆环缩小，/kg 倍数移至底部
- **颜色对齐**：蛋白质颜色统一为绿色，历史日历圆点颜色与表盘逻辑一致
- **目标值精确化**：从「倍数×体重」改为精确克数（力训日 139/98/109，休息日 98/98/109）
- **友好日期**：日期显示优化为「今天·周四」「昨天·周三」「5月27日·周二」格式

### Fixed
- **IndexedDB 版本冲突**：修复 DB_VERSION 降级导致的数据丢失问题，改用 localStorage 存储设置
- **导入兼容性**：支持导入 v1 和 v2 版本的备份文件
- **长按触发**：修复 MealGroup 条目长按无法触发多选的问题
- **临时食物显示**：最近 tab 隐藏临时食物（adhoc），只显示食物库中的食物
- **快捷添加体验**：打开快捷添加弹窗时自动聚焦份数输入框

### Technical
- **技术栈**：Vue 3 + TypeScript + Vite + Tailwind CSS + Pinia + IndexedDB (idb) + ECharts
- **部署**：GitHub Pages 自动部署（`npm run deploy`）
- **PWA**：vite-plugin-pwa 配置，支持离线使用
- **测试**：Vitest + jsdom + fake-indexeddb
